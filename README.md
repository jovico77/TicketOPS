# ¿Qué es TicketOPS?
Es una aplicación web para la gestión de incidencias desarrollada como proyecto de aprendizaje de DevOPS y Cloud.

## Objetivo
El objetivo del proyecto es utilizar todos los conocimientos que he adquirido tanto de ASIR, DAW y DevOPS para fortalezer al máximo estos conocimientos y en un futuro demostrarlos a travé de éste.
