# TicketOPS - DevOps Roadmap

## Objetivo

Este documento recoge la evolución técnica de TicketOPS.

El objetivo del proyecto no es únicamente desarrollar una aplicación Helpdesk, sino construir una plataforma moderna que permita aplicar y demostrar conocimientos profesionales de DevOps utilizados habitualmente en empresas.

Cada fase representa una tecnología o conjunto de conocimientos adquiridos durante el desarrollo del proyecto.

# Filosofía del proyecto

TicketOPS nace con un doble objetivo:

- Desarrollar una aplicación Helpdesk inspirada en escenarios reales.
- Construir una infraestructura moderna siguiendo las mejores prácticas DevOps.

El foco principal del proyecto será la automatización, la infraestructura y el despliegue continuo, utilizando la aplicación como plataforma para demostrar conocimientos profesionales.
