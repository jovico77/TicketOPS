## Conceptos importantes

### Migraciones

Versionan la estructura de la base de datos.

### Modelos

Representan tablas mediante Eloquent.

### Seeders

Insertan datos iniciales.

### Factories

Generan datos automáticamente.

### Fillable

Protege frente a asignación masiva.

### Relaciones

- belongsTo()
- hasMany()
